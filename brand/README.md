# Aura Media Group — Brand Identity

Complete visual identity system for **Aura Media Group** — *Your Marketing Growth Partner*.

## Files

| File | Description |
|------|-------------|
| `brand-identity.html` | Full brand book (13 sections) — open in any browser |
| `website-mockup.html` | Landing-page mockup (nav, hero, services, packages, footer) |
| `proposal.html` | Client proposal & pricing template (printable / PDF-ready) |
| `logo.svg` | Primary horizontal logo (mark + wordmark) |
| `logo-stacked.svg` | Stacked logo (mark above wordmark) |
| `logo-icon.svg` | Icon-only mark |
| `logo-avatar.svg` | Circular social avatar (Instagram / WhatsApp / LinkedIn) |
| `favicon.svg` | Favicon (navy rounded square) |

## How to view

Open `brand-identity.html` in any modern browser (no build step, fonts load from Google Fonts).

## What's inside the brand book

1. **Cover** — Logo lockup on Midnight Navy
2. **Logo System** — Light / dark / stacked / icon-only variants
3. **Color Palette** — Midnight Navy `#0F172A`, Champagne Gold `#D4B483`, White `#FFFFFF`, Soft Gray `#F8F8F8`
4. **Typography** — Montserrat (headlines) + Inter (body), all weights
5. **Business Card** — Front & back mockups
6. **Social Media** — Instagram profile mockup + post template
7. **Letterhead** — Branded A4 document
8. **Core Values** — Partnership · Creativity · Growth · Excellence · Transparency
9. **Key Messages** — Taglines (EN/AR) and core brand voice
10. **Vision & Mission** — Bilingual (EN/AR)
11. **Brand Personality** — Is / Is-Not chips
12. **Client Segments** — Aura Start · Aura Growth · Aura Personal
13. **Core Services** — Branding · Marketing · Content Creation · Website Development
14. **Content Pillars** — 6 pillars with example topics

## Brand colors (quick reference)

```
Midnight Navy    #0F172A    Primary — trust, leadership
Champagne Gold   #D4B483    Accent  — luxury, success
Pure White       #FFFFFF    Background — clarity
Soft Gray        #F8F8F8    Surface — neutral balance
```

## Typography

- **Headlines / Display:** Montserrat (also Poppins as alternate)
- **Body / UI:** Inter (also Open Sans as alternate)
